Linkurile de mai sus sunt ale site-urilor de unde au fost preluate si prelucrate informatiile utilizate in proiect.Scopul acestui proiect este de a servi drept material informativ asupra anumitor genuri muzicale cat si asupra anumitor muzicieni celebrii Design-ul proiectului este unul simplu, datorita faptului ca acesta trebuie sa fie usor de folosit de catre utilizatori. Navigatia intre genurile muzicale prezentate si muzicienii celebrii se poate realiza fie prin derularea in jos cu ajutorul mouse-ului fie prin apasare butonului triunghiular din partea stanga a header-ului care va afisa un meniu ce cuprinde materialele prezentate in proiect dar in mod individual, facilitand in mod semnificativ atat navigarea cat si experienta de navigare. Desi simplu, design-ul utilizat este unul modern, adapandu-se modului de vizualizare pe o gama larga de ecrane si rezolutii, putand fi utilizat fara probleme atat pe un dispozitiv desktop cat si pe o tableta sau un smartphone. Cantitatea de informatii prezentata nu este una exagerat de mare dar este suficienta pentru ca utilizatorii sa isi formuleze o idee despre generala despre continutul proiectului.


Tehnologiile realizate in scrierea proiectului sunt unele moderne, avand ultima versiune de HTML (ultima versiune in momentul realizarii acestui proiect), (HTML5), CSS3 si JavaScript (jQuery)


Sistemul de operare utilizat in scrierea proiectului este Microsoft Windows 10 iar intrag proiectul este scris in editorul Jetbrains Webstorm 2017.1.1
